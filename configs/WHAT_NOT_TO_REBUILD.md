# What Not To Rebuild

- Full IDE agent UI: use Cline or OpenHands.
- Browser automation engine: use Playwright MCP and Playwright tests.
- Git-aware patch engine: use Aider.
- Full autonomous dev platform: use OpenHands.
- GitHub pipeline system: use GitHub Actions and GitHub CLI.
- Model runtime: use Ollama.
- Claude Code protocol: use free-claude-code proxy.

## Dashboard Responsibilities

- Launch
- Monitor
- Gate
- Summarize
- Route
- Recover
- Display logs/reports
- Enforce safety
